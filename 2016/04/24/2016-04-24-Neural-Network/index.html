Machine Learning Course notes

Neural Network

Step 1 Pick a neural architecture

No. of input units: Dimensions of features x(i)

No. of output units: Number of classes

Reasonable default: 1 hidden layer, or if >1 hidden layer, have same no. of hidden units in every hidden layer (usually the more the better)

Step 2 Randomly initialize weights (否则计算出的隐藏层各节点结果相同）

Step 3 Implement forward propagation to get hθ(x(i)) for any x(i)

Step 4 Implement code to compute cost function J(θ)

Cost function

Step 5 Implement backpropagation to compute partial derivatives

Step 6 Use gradient checking to compare from backpropagation vs. using numerical estimate of derivatives of ,then disable gradient checking code

Step 7 Use gradient decent or advanced optimization with backpropagation to try to minimize as a function of parameter
